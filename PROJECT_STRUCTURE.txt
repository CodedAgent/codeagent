KEY MODULES EXPLAINED:

CLI Layer (cli.rs)
  - Handles command-line argument parsing
  - Two commands: `run` (execute task) and `init` (initialize project)
  - Supports: prompts, dry-run mode, help documentation

Core Engine (core/)
  - config.rs: Loads/manages model provider configuration
  - executor.rs: Main task execution orchestrator
  - planner.rs: Converts prompts into execution plans
  - context.rs: Analyzes project structure and context

Integrations (integrations/)
  - git.rs: Git operations (stage, commit, status)
  - ollama.rs: Async client for Ollama LLM API
  - test_runner.rs: Detects and runs multiple test frameworks

Utilities (utils/)
  - file_utils.rs: File discovery, read/write operations
  - search.rs: Regex and keyword-based search across files
