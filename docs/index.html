Installation v0.3.0

macOS Installation

Download the latest release for your architecture:

# Intel (x86_64)
curl -L https://github.com/yourusername/CodeAgent/releases/download/v0.3.0/codeagent-macos-x86_64 -o codeagent
chmod +x codeagent
./codeagent --version

# Apple Silicon (ARM64)
curl -L https://github.com/yourusername/CodeAgent/releases/download/v0.3.0/codeagent-macos-aarch64 -o codeagent
chmod +x codeagent
./codeagent --version

Optional: Add to PATH

sudo mv codeagent /usr/local/bin/
codeagent --help

Linux Installation

Download the Linux x86_64 binary:

curl -L https://github.com/yourusername/CodeAgent/releases/download/v0.3.0/codeagent-linux-x86_64 -o codeagent

Windows Installation

Download the Windows x86_64 binary:

# Using PowerShell
$url = "https://github.com/yourusername/CodeAgent/releases/download/v0.3.0/codeagent-windows-x86_64"
Invoke-WebRequest -Uri $url -OutFile codeagent.exe
.\codeagent.exe --version

Optional: Add to System PATH for global access

Build from Source

Requirements: Rust 1.70+

# Clone the repository
git clone https://github.com/yourusername/CodeAgent.git
cd CodeAgent

# Build release binary
cargo build --release

# Run
./target/release/codeagent --help
